**Fix E — `PhotoUploadGrid.tsx`: fix stale slots count**

File: `components/onboarding/PhotoUploadGrid.tsx`

The `onPhotoCountChange` callback reads stale `slots` state. Fix by computing
count inside the state setter:

In `pickAndUpload`, replace:
```typescript
updateSlot(order, { url: data.publicUrl, storagePath: path, uploading: false });
onPhotoCountChange?.(slots.filter(s => s.url !== null).length + 1);
```
With:
```typescript
setSlots(prev => {
  const next = prev.map(s =>
    s.order === order ? { ...s, url: data.publicUrl, storagePath: path, uploading: false } : s
  );
  onPhotoCountChange?.(next.filter(s => s.url !== null).length);
  return next;
});
```

In `removePhoto`, replace:
```typescript
updateSlot(order, { url: null, storagePath: null, uploading: false, error: false });
onPhotoCountChange?.(slots.filter(s => s.url !== null && s.order !== order).length);
```
With:
```typescript
setSlots(prev => {
  const next = prev.map(s =>
    s.order === order ? { ...s, url: null, storagePath: null, uploading: false, error: false } : s
  );
  onPhotoCountChange?.(next.filter(s => s.url !== null).length);
  return next;
});
```

**Fix F — `basics.tsx`: get real profileId before passing to PhotoUploadGrid**

File: `app/(onboarding)/basics.tsx`

The current code passes `userId` (auth UID) as `profileId` to `PhotoUploadGrid`,
but `profile_photos.profile_id` is a FK to `profiles.id` (a different UUID).
This will cause a FK violation when photos are uploaded.

Fix: after the profile upsert in `handleSubmit` succeeds, fetch the profile `id`
and store it in state. Then use that profileId for PhotoUploadGrid.

Add a state variable:
```typescript
const [profileId, setProfileId] = useState<string | null>(null);
```

After the profile upsert (in handleSubmit), fetch the profile id:
```typescript
const { data: profileRow } = await supabase
  .from('profiles')
  .select('id')
  .eq('user_id', userId!)
  .single();
if (profileRow) setProfileId(profileRow.id);
```

Update the PhotoUploadGrid render:
```typescript
{profileId ? (
  <PhotoUploadGrid profileId={profileId} />
) : userId ? (
  <Text style={{ color: Colors.textMuted, fontSize: 12 }}>
    save your basics first to add photos
  </Text>
) : null}
```

After all fixes, run `npx tsc --noEmit` to confirm clean compile, run
`npx jest --no-coverage` to confirm tests pass, then commit:
```bash
git add -A
git commit -m "fix: quality issues from code review — slots count, FiltersSheet reset, ButWhy deps, OverlayIndicator top, SwipeCard clamp, basics profileId"
```
